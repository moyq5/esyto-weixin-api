# 删除自动回复规则

**服务地址：**

| 服务地址 | http:\/\/localhost:8888\/v2.0\/wx\/reply\/{replyId} |
| --- | --- |
| 请求方式 | DELETE |

**请求参数：**

> 示例

```js
N/A
```

> 字段说明

| **字段** | **含义** | **类型** | **必填** | **描述** |
| --- | --- | --- | --- | --- |
| replyId | 回复规则记录ID | long\(14\) | 是 | 用“,”号隔开多个ID可实现批量删除 |

**响应参数：**

> 示例

```js
{
  "response_text" : {
    "code" : "0",
    "message" : "Success"
  }
}
```

> 字段说明

| **字段** | **含义** | **类型** | **必填** | **描述** |
| --- | --- | --- | --- | --- |
| code | 错误码 | string\(8\) | 是 | 等于0时表示成功，否则表示失败 |
| message | 描述信息 | string\(200\) | 是 | 成功或者失败原因描述 |
